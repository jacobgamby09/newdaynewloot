# New Day New Loot Docs

This folder contains the current design guidance for the mining incremental prototype.

Read in this order:

1. `GDD.md` - overall game design and agreed direction.
2. `visual.md` - art direction, tile style, palette, and sprite guidance.
3. `VISUAL_FEEL_GUIDE.md` - camera, animation, UI, and payoff guidance.
4. `SYSTEMS_GUIDE.md` - mechanics and progression systems.
5. `MVP_ROADMAP.md` - recommended implementation phases.

The current direction is a short-session incremental mining game where runs stay compact, workers clear a visible tile-based mine, resources fly into the UI, and camp upgrades drive long-term progression.
